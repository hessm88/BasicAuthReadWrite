UserEndpoint

Endpoint for basic user account functionality

The following resources are applicable:

  • /name/create
  • /name/name

/name/create

Mount Point: /rest/name/create

GET

Create a new user with the given characteristics. This function does NOT log the new user in.

Parameters

name description type default
email the new email to create a user account for query
password the password to associate with the email query
last-name the last name of the new user query
first-name the first name of the new user query

Response Body

element: (custom)
media types: */*
application/xml
application/json

(no documentation provided)

Status Codes

HTTP Status Code Description
200 OK User was created.
403 Forbidden Account already associated with provided email address
404 Not Found Email and/or password is/are incorrect

/name/name

Mount Point: /rest/name/name

GET

Get the name of the user associated with the email. Note that a user can only retrieve their own name, i.e. not the names of any other users.

Parameters

name description type default
email the email associated with the user query
session-id the session ID assigned to the user query

Response Body

element: (custom)
media types: */*
application/xml
application/json

(no documentation provided)

Status Codes

HTTP Status Code Description
200 OK User for user returned
403 Forbidden User's session ID is not correct
500 Internal Server Error User has validated and should have a name returned, but no name was returned
